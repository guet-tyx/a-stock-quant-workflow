# 个股多因子选股策略研究报告（修复版）

## 执行摘要

本研究构建了一个基于A股的多因子选股策略，目标是超越已有的ETF v7.0策略。研究发现：

1. **小市值因子**是最强的预测因子（IC=0.1977，IC_IR=1.7940）
2. **低波动因子**也呈现正向有效性（IC=0.0734，IC_IR=0.9257）
3. **ROE因子**呈现反向有效性（IC=-0.1543，IC_IR=-2.3113）
4. 策略回测显示年化收益18.17%，夏普比率3.78，最大回撤-21.57%

## 1. 研究背景

### 1.1 研究目标
- **核心目标**：构建A股多因子选股策略
- **收益目标**：年化收益15%+（超越ETF策略的11.30%）
- **风险目标**：夏普比率>1.0，最大回撤<20%

## 4. 策略构建

### 4.1 因子选择
基于IC分析，选择以下6个因子构建策略：

1. **小市值因子**（正向，IC_IR=1.7940）
2. **流动性因子**（正向，IC_IR=1.7940）
3. **低波动因子**（正向，IC_IR=0.9257）
4. **ROE因子**（反向，IC_IR=-2.3113，需取负）
5. **净利率因子**（反向，IC_IR=-1.2802，需取负）
6. **现金流/负债因子**（反向，IC_IR=-0.4376，需取负）

### 4.2 因子权重
使用IC_IR加权：

| 因子 | IC_IR | 权重 |
|------|-------|------|
| 小市值因子 | 1.7940 | 21.0% |
| 流动性因子 | 1.7940 | 21.0% |
| 低波动因子 | 0.9257 | 10.8% |
| ROE因子 | -2.3113 | -27.1% |
| 净利率因子 | -1.2802 | -15.0% |
| 现金流/负债因子 | -0.4376 | -5.1% |

### 4.3 选股规则
- **选股数量**：Top 50只（等权重）
- **行业约束**：单行业上限15%
- **调仓频率**：月度（每月最后一个交易日）

### 4.4 风险控制
- **止损**：单只个股亏损>10%强制卖出
- **仓位上限**：单只个股不超过总资金3%
- **最大回撤控制**：回撤>10%时降仓至50%
