# A股个股多因子量化策略工作流

## 📊 项目概述

本项目是一个完整的A股个股多因子量化策略研究框架，包含因子挖掘、策略构建、回测验证等全流程。

### ⚠️ 重要说明

**本仓库数据已消除前视偏差**。之前的版本（年化46.60%，夏普1.56）存在严重前视偏差，修复后真实alpha大幅缩水。详细过程见下方。

## ⚙️ 策略配置（v2最优）

```yaml
top_n: 30                    # 持仓30只
max_turnover: 0.3            # 月换手≤30%
train_window: 24             # 训练窗口24月
retrain_freq: 6              # 每6月重训
transaction_cost: 0.007      # 来回成本0.70%
stop_loss_individual: -0.08  # 个股止损-8%
stop_loss_portfolio: -0.12   # 组合止损-12%
```

## 📊 数据说明

- **数据源**: Tushare，**股票池**: 沪深300+中证500（约515只）
- **时间范围**: 2021-01 ~ 2026-06，**频率**: 日线，月度调仓
- **回测方法**: Walk-Forward（24月训练+6月重训）
- **前视偏差审计**: 三轮审计（静态检查+运行时验证+时间线检查）
- **因子去重**: 44→11个（去重后夏普提升30%）

## 🚨 v1→v2 修复对比

| 指标 | v1（修复前） | v2（修复后） | 原因 |
|------|------------|------------|------|
